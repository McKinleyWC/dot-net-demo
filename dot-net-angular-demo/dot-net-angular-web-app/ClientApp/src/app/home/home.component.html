Welcome to my single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

For this demo, I decided to use the Annual Production Report data from the Dept. of the Treasury. The data is consumed from the API, database tables are cleared, and the data is inserted. I considered using Entity Framework for this, but I'm not familiar with it beyond it's general use and didn't want to try to use too many unfamiliar things in a demo. In a real world scenario, I would learn to use Entity.

The treasury API client and Sql utilities are in a DAL project using Dapper, Newtonsoft, and Linq.
